Quick Start

Get up and running with goscapy in 5 minutes. This guide covers installation, packet construction, dissection, sending, and sniffing.

Installation

Add goscapy to your project:

bash 
go get github.com/smallnest/goscapy

Import the packages you need:

go 
import (
    "fmt"
    "time"

    "github.com/smallnest/goscapy/pkg/goscapy"
    "github.com/smallnest/goscapy/pkg/packet"
    "github.com/smallnest/goscapy/pkg/sendrecv"
    "github.com/smallnest/goscapy/pkg/sniff"
)
📝 Note
Raw socket operations require root/administrator privileges. Run your programs with sudo on Linux or macOS.

Build a Packet (Builder API)

The Builder API provides fluent method chaining for type-safe, explicit packet construction. Each protocol layer has its own builder with methods for setting fields. Chain layers together with .Over() and finalize with .Build().

Ethernet + IP + ICMP

go 
// Build an ICMP Echo Request packet from scratch
pkt, err := goscapy.NewEthernet().
    SrcMAC("aa:bb:cc:dd:ee:ff").
    DstMAC("ff:ff:ff:ff:ff:ff").
    Over(goscapy.NewIP().
        SrcIP("192.168.1.1").
        DstIP("8.8.8.8").
        TTL(64)).
    Over(goscapy.NewICMP().
        Type(8).   // Echo Request
        Code(0).
        ID(1).
        Seq(1)).
    Build()
if err != nil {
    log.Fatal(err)
}
fmt.Printf("Built %d bytes\n", len(pkt))

Ethernet + IP + TCP

go 
// Build a TCP SYN packet
pkt, err := goscapy.NewEthernet().
    SrcMAC("aa:bb:cc:dd:ee:ff").
    DstMAC("11:22:33:44:55:66").
    Over(goscapy.NewIP().
        SrcIP("192.168.1.100").
        DstIP("93.184.216.34")).
    Over(goscapy.NewTCP().
        SrcPort(12345).
        DstPort(80).
        Flags(0x02).  // SYN
        Seq(1000).
        Window(65535)).
    Build()

Ethernet + IP + UDP

go 
// Build a UDP packet
pkt, err := goscapy.NewEthernet().
    SrcMAC("aa:bb:cc:dd:ee:ff").
    DstMAC("11:22:33:44:55:66").
    Over(goscapy.NewIP().
        SrcIP("192.168.1.100").
        DstIP("8.8.8.8")).
    Over(goscapy.NewUDP().
        SrcPort(54321).
        DstPort(53)).
    Build()
💡 Tip
Checksums (IP, TCP, UDP, ICMP) are calculated automatically during .Build(). You never need to set them manually.

Build a Packet (Shortcuts)

Shortcut functions build common protocol stacks in a single call. They use sensible defaults (e.g. srcMAC="00:00:00:00:00:00", srcIP="0.0.0.0") so you only specify what matters.

go 
// ICMP Echo Request — one line
pkt, err := goscapy.EtherIPICMP("ff:ff:ff:ff:ff:ff", "8.8.8.8", 8, 0)

// TCP SYN packet
pkt, err := goscapy.EtherIPTCP(
    "aa:bb:cc:dd:ee:ff", "11:22:33:44:55:66",
    "192.168.1.100", "93.184.216.34",
    12345, 80, 0x02,
)

// UDP packet
pkt, err := goscapy.EtherIPUDP(
    "aa:bb:cc:dd:ee:ff", "11:22:33:44:55:66",
    "192.168.1.100", "8.8.8.8",
    54321, 53,
)

// ARP request
pkt, err := goscapy.EtherARP(
    "aa:bb:cc:dd:ee:ff", "ff:ff:ff:ff:ff:ff",
    "192.168.1.100", "192.168.1.1",
    1, // ARP Request
)

// IP-only packets (no Ethernet header)
pkt, err := goscapy.IPICMP("192.168.1.100", "8.8.8.8", 8, 0)
pkt, err := goscapy.IPTCP("192.168.1.100", "93.184.216.34", 12345, 80, 0x02)
pkt, err := goscapy.IPUDP("192.168.1.100", "8.8.8.8", 54321, 53)

Dissect a Packet

Parse raw bytes back into a structured Packet object. goscapy auto-detects the protocol at each layer and extracts all fields. Use DissectEthernet for full frames or DissectIP for IP-only data.

go 
// First, build a packet to dissect
raw, _ := goscapy.EtherIPICMP("ff:ff:ff:ff:ff:ff", "8.8.8.8", 8, 0)

// Dissect it back into layers
pkt, err := packet.Dissect(raw, packet.DissectEthernet)
if err != nil {
    log.Fatal(err)
}

// Print the layer stack
fmt.Println(pkt.String()) // "Ethernet / IP / ICMP"

// Access individual layers
eth := packet.Ethernet(pkt)
fmt.Printf("SrcMAC: %s\n", eth.SrcMAC())
fmt.Printf("DstMAC: %s\n", eth.DstMAC())

ip := packet.IP(pkt)
fmt.Printf("SrcIP: %s  DstIP: %s  TTL: %d\n",
    ip.SrcIP(), ip.DstIP(), ip.TTL())
💡 Tip
Use packet.DissectIP instead of packet.DissectEthernet when your raw bytes start at the IP header (no Ethernet frame).

Send a Packet

goscapy supports sending at two levels:

  • L3 (Send) — sends at the IP level. The OS adds the Ethernet header. Use for IP+Transport packets.
  • L2 (Sendp) — sends the full Ethernet frame. You control the entire packet.
go 
// Build an ICMP Echo Request
raw, _ := goscapy.EtherIPICMP("ff:ff:ff:ff:ff:ff", "8.8.8.8", 8, 0)
pkt, _ := packet.Dissect(raw, packet.DissectEthernet)

// Send at L3 — OS handles Ethernet framing
err := sendrecv.Send(pkt, "eth0")

// Send at L2 — you provide the full frame
err = sendrecv.Sendp(pkt, "eth0")

Send and Receive a Response

Use SendRecv to send a packet and collect responses, or SendRecv1 to get just the first response:

go 
// Send ICMP Echo and wait for reply
raw, _ := goscapy.EtherIPICMP("ff:ff:ff:ff:ff:ff", "8.8.8.8", 8, 0)
pkt, _ := packet.Dissect(raw, packet.DissectEthernet)

// Get all responses within 3 seconds
sent, responses, err := sendrecv.SendRecv(pkt, "eth0", 3*time.Second)
fmt.Printf("Got %d responses\n", len(responses))

// Or just get the first response
sent, reply, err := sendrecv.SendRecv1(pkt, "eth0", 3*time.Second)
if reply != nil {
    fmt.Println("Reply:", reply.String())
}
📝 Note
SendRecv opens the receiver before sending to avoid missing fast responses (e.g. on loopback).

Sniff Packets

Capture live traffic using either a callback handler or a channel. Both support BPF filter expressions (requires tcpdump on PATH for compilation).

Callback-Based Sniffing

go 
// Sniff ICMP packets for 10 seconds
err := sniff.Sniff(sniff.SniffConfig{
    Iface:   "eth0",
    Filter:  "icmp",
    Timeout: 10 * time.Second,
}, func(pkt *packet.Packet) bool {
    fmt.Println(pkt.String())
    return true // continue sniffing
})

Channel-Based Sniffing

go 
// Capture exactly 5 TCP packets
ch, stop := sniff.SniffChan(sniff.SniffConfig{
    Iface:  "eth0",
    Filter: "tcp",
    Count:  5,
})
defer stop()

for pkt := range ch {
    fmt.Println(pkt.String())
}
📝 Note
BPF filters are compiled at kernel level for maximum performance. Use sniff.CompileFilter("tcp port 80") to pre-compile filter strings.
