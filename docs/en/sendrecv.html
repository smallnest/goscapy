Platform Compatibility Matrix

The sendrecv package provides standard raw socket capabilities with advanced high-performance features optimized for Linux systems.

Feature / API macOS (Darwin) Linux Requirements & Kernel Options
RawConn (DialRaw) ✅ Supported ✅ Supported Requires root privileges (raw socket permission)
SendRaw / RecvRaw ✅ Supported ✅ Supported Package-level stateless convenience functions; requires root
BatchConn (SendBatch / RecvBatch) ⚠️ Sequential Fallback ✅ Supported Uses sendmmsg and recvmmsg system calls under Linux
ZeroCopy (SetZeroCopy / MSG_ZEROCOPY) ❌ Returns ErrNotSupported ✅ Supported Linux 4.14+ kernels; sets SO_ZEROCOPY socket option
UringConn (io_uring Async IO) ❌ Returns ErrNotSupported ✅ Supported Linux 5.1+ kernels; maps Submission & Completion queues
PacketMMAP (TPACKET_V3 Capture) ❌ Returns ErrNotSupported ✅ Supported Requires AF_PACKET raw socket mapping (Linux only)

Raw Socket Connection (RawConn)

RawConn manages a connection-oriented raw socket. Created using the DialRaw function.

DialRaw

go
func DialRaw(proto int) (*RawConn, error)

Opens a raw connection for the specified IP protocol number (e.g. 1 for ICMP, 6 for TCP, 17 for UDP). Returns an error if raw socket creation fails (e.g. insufficient privileges).

Send

go
func (c *RawConn) Send(payload []byte, dst string) error

Sends the raw payload (without Ethernet header) to the target destination IP address. If ZeroCopy is enabled, it sends asynchronously and marks the payload memory until completion is monitored.

Recv

go
func (c *RawConn) Recv(timeout time.Duration) ([]byte, string, error)

Blocks until a packet arrives or the specified timeout is reached. Returns the raw payload bytes, source IP address, and an error. If the timeout is exceeded, it returns sendrecv.ErrTimeout.

Batch

go
func (c *RawConn) Batch() *BatchConn

Returns a BatchConn wrapper bound to this raw connection to perform multi-packet IO operations.

SetZeroCopy

go
func (c *RawConn) SetZeroCopy(enable bool) error

Enables or disables zero-copy transmitting. On Linux, this enables SO_ZEROCOPY on the socket. On macOS, this returns sendrecv.ErrNotSupported.

WaitZeroCopyCompletion

go
func (c *RawConn) WaitZeroCopyCompletion(ctx context.Context) error

Waits for the Linux kernel to release all pending zero-copy buffers. Returns an error if the context expires before completions are received.

Close

go
func (c *RawConn) Close() error

Closes the raw socket file descriptor and releases system resources.

Stateless Convenience API

For simple operations where connection reuse is not required, use these package-level functions.

SendRaw

go
func SendRaw(proto int, payload []byte, dst string) error

Opens a temporary raw socket, transmits the payload to the target IP, and immediately closes the socket.

RecvRaw

go
func RecvRaw(proto int, timeout time.Duration) ([]byte, string, error)

Opens a temporary raw socket, blocks until a packet arrives or timeout is exceeded, and closes the socket.

Batch Operations (BatchConn)

BatchConn leverages Linux system calls sendmmsg and recvmmsg to perform batch sending and receiving, significantly reducing kernel-user mode transitions.

SendBatch

go
func (b *BatchConn) SendBatch(msgs []BatchMsg) (int, error)

Sends multiple payloads in a single batch. On macOS, this falls back to a sequential sending loop. Returns the number of successfully transmitted packets.

go
type BatchMsg struct {
    Data []byte
    Dst  string
}

RecvBatch

go
func (b *BatchConn) RecvBatch(n int, timeout time.Duration) ([]BatchResult, error)

Attempts to read up to n packets within the specified timeout. Returns a slice of BatchResult messages.

go
type BatchResult struct {
    Data  []byte
    SrcIP string
    Err   error
}

io_uring Socket (UringConn)

UringConn utilizes the modern Linux io_uring subsystem. By submitting asynchronous socket read/write requests directly to submission queues (SQ) and polling completion queues (CQ), it provides maximum IO throughput and eliminates context switching.

DialUringRaw

go
func DialUringRaw(proto int) (*UringConn, error)

Dials a raw socket using a mapped io_uring queue on Linux. Returns sendrecv.ErrNotSupported on non-Linux platforms or Linux kernels older than 5.1.

Send

go
func (u *UringConn) Send(payload []byte, dst string) (uint64, error)

Enqueues a non-blocking raw packet write to the Submission Queue (SQ). Returns a unique operation ID which corresponds to the kernel's completion event.

Recv

go
func (u *UringConn) Recv(timeout time.Duration) ([]byte, string, error)

Polls the Completion Queue (CQ) or waits for a read operation to complete within the specified timeout.

SendRecvBatch

go
func (u *UringConn) SendRecvBatch(msgs []BatchMsg) ([]BatchResult, error)

Submits a batch of write and read events to the ring buffer in a single operation.

Close

go
func (u *UringConn) Close() error

Tears down the submission and completion rings and closes the associated raw socket file descriptor.

Packet Capture Ring Buffer (PacketMMAP)

PacketMMAP implements a high-performance network sniffer by memory-mapping (mmap) a ring buffer structure (TPACKET_V3) over an AF_PACKET raw socket on Linux.

NewPacketMMAP

go
func NewPacketMMAP(iface string) (*PacketMMAP, error)

Maps an AF_PACKET capture ring on the specified interface (e.g. eth0). Returns sendrecv.ErrNotSupported on macOS.

Recv

go
func (m *PacketMMAP) Recv(timeout time.Duration) (*packet.Packet, error)

Retrieves the next packet from the memory ring buffer. It automatically parses and dissects the payload using the goscapy parser starting from the link layer.

RecvBatch

go
func (m *PacketMMAP) RecvBatch(n int, timeout time.Duration) ([]*packet.Packet, error)

Reads up to n packets from the mapped memory area.

Stats

go
func (m *PacketMMAP) Stats() PacketMMAPStats

Retrieves real-time packet statistics (received packets, dropped packets, queue freezes) directly from the socket option (TPACKET_STATS_V3).

go
type PacketMMAPStats struct {
    Received uint32
    Dropped  uint32
    Freeze   uint32
}

Close

go
func (m *PacketMMAP) Close() error

Unmaps the ring memory and releases the packet socket interface.
