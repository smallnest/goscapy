Ethernet + IP + GRE + Inner IP

Build a GRE tunnel packet encapsulating an inner IP payload. GRE is commonly used for site-to-site VPNs and overlay networks. Setting .Key(), .Seq(), or .SetChecksum() automatically enables the corresponding GRE flags (K, S, C bits).

🔨 Builder API

go
import (
    "github.com/smallnest/goscapy/pkg/goscapy"
    "github.com/smallnest/goscapy/pkg/layers/gre"
)

// Build an inner IP packet to encapsulate
innerPayload, _ := goscapy.NewIP().
    SrcIP("10.1.0.1").
    DstIP("10.2.0.1").
    Over(goscapy.NewICMP().
        Type(8).Code(0)).
    Build()

// Outer GRE tunnel: Ethernet + IP + GRE + Inner IP
raw, err := goscapy.NewEthernet().
    SrcMAC("00:11:22:33:44:55").
    DstMAC("66:77:88:99:aa:bb").
    Over(goscapy.NewIP().
        SrcIP("192.168.1.1").
        DstIP("192.168.2.1")).
    Over(goscapy.NewGRE().
        ProtocolType(gre.ProtoIP).       // 0x0800 = encapsulated IP
        Key(0x00001000).                   // sets K flag automatically
        Seq(42).                          // sets S flag automatically
        Payload(innerPayload)).            // inner IP packet as raw bytes
    Build()

⚡ Shortcut

go
// EtherIPGRE(srcMAC, dstMAC, srcIP, dstIP string, protoType uint16, key uint32, innerPayload []byte)
// key=0 means no Key field; non-zero sets K flag automatically
raw, err := goscapy.EtherIPGRE(
    "00:11:22:33:44:55", "66:77:88:99:aa:bb",
    "192.168.1.1", "192.168.2.1",
    gre.ProtoIP,       // ProtocolType = 0x0800 (IP)
    0x00001000,        // Key value
    innerPayload,      // inner IP + ICMP bytes
)

🔬 Dissect

go
import "github.com/smallnest/goscapy/pkg/packet"

pkt, err := packet.Dissect(raw, packet.DissectEthernet)
fmt.Println(pkt.String()) // "Ethernet / IP / GRE"

// Access GRE fields
greLayer := packet.Layer(pkt, "GRE")
fmt.Printf("ProtoType=0x%04x  HasKey=%v  HasSeq=%v\n",
    gre.GetProtocolType(greLayer),
    gre.HasKey(greLayer),
    gre.HasSeq(greLayer))
if gre.HasKey(greLayer) {
    fmt.Printf("Key=0x%08x\n", gre.GetKey(greLayer))
}
if gre.HasSeq(greLayer) {
    fmt.Printf("Seq=%d\n", gre.GetSeq(greLayer))
}
Note: GRE conditional fields (Key, Seq, Checksum) are only included in the serialized output when explicitly set. Setting a field via the Builder automatically enables the corresponding flag bit in the GRE header.

GRE with Checksum Field

GRE supports an optional checksum for integrity verification. Setting .SetChecksum() enables the C flag and includes the checksum field in the header. When using .Build(), the checksum value can be set to 0 — it will be computed automatically during serialization.

🔨 Builder API

go
// GRE with all optional fields: Checksum + Key + Sequence
raw, err := goscapy.NewEthernet().
    SrcMAC("00:11:22:33:44:55").
    DstMAC("66:77:88:99:aa:bb").
    Over(goscapy.NewIP().
        SrcIP("192.168.1.1").
        DstIP("192.168.2.1")).
    Over(goscapy.NewGRE().
        ProtocolType(gre.ProtoEthernet).  // 0x6558 = encapsulated Ethernet
        SetChecksum(0).                   // auto-computed; enables C flag
        Key(0xABCD0001).
        Seq(100).
        Payload(innerEthernetFrame)).
    Build()

📦 GRE Header Layout

text
// GRE Header with C+K+S flags set (variable length)
//
//   Flags (2 bytes): C=1, R=0, K=1, S=1
//   ProtocolType (2 bytes): 0x6558 (Ethernet) or 0x0800 (IP)
//   Checksum + Reserved1 (4 bytes) — only if C=1
//   Key (4 bytes) — only if K=1
//   Sequence (4 bytes) — only if S=1
//
// Minimum GRE header = 4 bytes (flags + protocol type)
// Maximum with all optional fields = 16 bytes

Ethernet + IP + UDP + VXLAN + Inner Ethernet

Build a VXLAN tunnel packet — the de-facto standard for network virtualization in data centers. VXLAN encapsulates a Layer 2 Ethernet frame within a UDP datagram, identified by a 24-bit VNI (VXLAN Network Identifier).

🔨 Builder API

go
// Inner Ethernet frame to encapsulate
innerFrame, _ := goscapy.NewEthernet().
    SrcMAC("aa:bb:cc:dd:ee:01").
    DstMAC("aa:bb:cc:dd:ee:02").
    Over(goscapy.NewIP().
        SrcIP("10.1.0.1").
        DstIP("10.1.0.2")).
    Over(goscapy.NewTCP().
        SrcPort(12345).DstPort(80)).
    Build()

// Outer VXLAN tunnel
raw, err := goscapy.NewEthernet().
    SrcMAC("00:11:22:33:44:55").
    DstMAC("66:77:88:99:aa:bb").
    Over(goscapy.NewIP().
        SrcIP("192.168.1.1").
        DstIP("192.168.2.1")).
    Over(goscapy.NewUDP().
        SrcPort(4789).       // standard VXLAN UDP port
        DstPort(4789)).
    Over(goscapy.NewVXLAN().
        VNI(100500).             // 24-bit VXLAN Network Identifier
        Payload(innerFrame)).      // inner Ethernet frame
    Build()

⚡ Shortcut

go
// EtherIPUDPVXLAN(srcMAC, dstMAC, srcIP, dstIP string, vni uint32, innerPayload []byte)
// Uses UDP port 4789 for both src and dst by default
raw, err := goscapy.EtherIPUDPVXLAN(
    "00:11:22:33:44:55", "66:77:88:99:aa:bb",
    "192.168.1.1", "192.168.2.1",
    100500,          // VNI
    innerFrame,      // inner Ethernet frame bytes
)

🔬 Dissect

go
import "github.com/smallnest/goscapy/pkg/layers/vxlan"

pkt, err := packet.Dissect(raw, packet.DissectEthernet)
fmt.Println(pkt.String()) // "Ethernet / IP / UDP / VXLAN"

// Access VXLAN fields
vxlanLayer := packet.Layer(pkt, "VXLAN")
fmt.Printf("VNI=%d  Flags=0x%02x\n",
    vxlan.GetVNI(vxlanLayer),
    vxlan.GetFlags(vxlanLayer))
Tip: The VXLAN VNI is a 24-bit value (0–16,777,215). The I flag (bit 3 of the flags byte) is always set in valid VXLAN packets and is handled automatically by the builder.

Ethernet + Dot1Q (VLAN) + IP

Build an 802.1Q VLAN-tagged packet. Dot1Q inserts a 4-byte shim between the Ethernet header and the payload, carrying the VLAN ID (VID), Priority Code Point (PCP), and Drop Eligible Indicator (DEI).

🔨 Builder API

go
import "github.com/smallnest/goscapy/pkg/layers/dot1q"

raw, err := goscapy.NewEthernet().
    SrcMAC("00:11:22:33:44:55").
    DstMAC("ff:ff:ff:ff:ff:ff").
    Over(goscapy.NewDot1Q().
        VID(100).            // VLAN ID (0–4095)
        PCP(5).             // Priority Code Point (0–7)
        DEI(false).          // Drop Eligible Indicator
        Type(0x0800)).      // inner EtherType = IPv4
    Over(goscapy.NewIP().
        SrcIP("192.168.1.1").
        DstIP("10.0.0.1")).
    Over(goscapy.NewTCP().
        SrcPort(12345).DstPort(80)).
    Build()

⚡ Shortcut

go
// EtherDot1QIP(srcMAC, dstMAC, srcIP, dstIP string, vid uint16)
// Default: PCP=0, DEI=false, Type=0x0800 (IPv4), TPID=0x8100
raw, err := goscapy.EtherDot1QIP(
    "00:11:22:33:44:55", "ff:ff:ff:ff:ff:ff",
    "192.168.1.1", "10.0.0.1",
    100,  // VLAN ID
)

🔬 Dissect

go
pkt, err := packet.Dissect(raw, packet.DissectEthernet)
fmt.Println(pkt.String()) // "Ethernet / Dot1Q / IP / TCP"

// Access Dot1Q fields
dot1qLayer := packet.Layer(pkt, "Dot1Q")
fmt.Printf("VID=%d  PCP=%d  DEI=%v  Type=0x%04x\n",
    dot1q.GetVID(dot1qLayer),
    dot1q.GetPCP(dot1qLayer),
    dot1q.GetDEI(dot1qLayer),
    dot1q.GetType(dot1qLayer))

📦 Dot1Q Header Layout (4 bytes)

text
// Dot1Q 4-byte tag inserted after Ethernet src/dst MAC
//
//   TPID (2 bytes): 0x8100 (802.1Q) or 0x88A8 (802.1ad/QinQ)
//   TCI (2 bytes):
//     PCP  (3 bits) — Priority Code Point (0–7)
//     DEI  (1 bit)  — Drop Eligible Indicator
//     VID (12 bits) — VLAN Identifier (0–4095)

QinQ — Double VLAN Tagging

QinQ (802.1ad) stacks two VLAN tags — an outer service tag (S-tag) with TPID 0x88A8 and an inner customer tag (C-tag) with TPID 0x8100. This allows service providers to tunnel customer VLANs transparently.

🔨 Builder API

go
raw, err := goscapy.NewEthernet().
    SrcMAC("00:11:22:33:44:55").
    DstMAC("ff:ff:ff:ff:ff:ff").
    Over(goscapy.NewDot1Q().           // Outer S-tag (service)
        TPID(dot1q.TPID8021AD).      // 0x88A8 = 802.1ad
        VID(2000).
        PCP(3).
        DEI(false)).
    Over(goscapy.NewDot1Q().           // Inner C-tag (customer)
        TPID(dot1q.TPID8021Q).       // 0x8100 = 802.1Q
        VID(100).
        PCP(0).
        DEI(false)).
    Over(goscapy.NewIP().
        SrcIP("192.168.1.1").
        DstIP("10.0.0.1")).
    Over(goscapy.NewICMP().
        Type(8).Code(0)).
    Build()

🔬 Dissect

go
pkt, err := packet.Dissect(raw, packet.DissectEthernet)
fmt.Println(pkt.String()) // "Ethernet / Dot1Q / Dot1Q / IP / ICMP"

// Access outer S-tag
outerTag := packet.Layer(pkt, "Dot1Q")
fmt.Printf("S-Tag: VID=%d  PCP=%d\n",
    dot1q.GetVID(outerTag), dot1q.GetPCP(outerTag))

// Access inner C-tag (second Dot1Q layer)
dot1qLayers := packet.Layers(pkt, "Dot1Q")
if len(dot1qLayers) > 1 {
    innerTag := dot1qLayers[1]
    fmt.Printf("C-Tag: VID=%d  PCP=%d\n",
        dot1q.GetVID(innerTag), dot1q.GetPCP(innerTag))
}
Note: The outer Dot1Q uses TPID8021AD (0x88A8) to distinguish it from a single-tagged frame. The inner tag uses the standard TPID8021Q (0x8100). The dissection engine automatically recognizes nested Dot1Q layers.
