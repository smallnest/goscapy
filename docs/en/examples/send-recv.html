Send — ICMP Echo Request (L3)

Build an IP+ICMP packet and send it at L3. The OS adds the Ethernet header automatically. Send() is fire-and-forget — it does not wait for a response.

🔨 Builder API

go
import (
    "fmt"
    "github.com/smallnest/goscapy/pkg/goscapy"
    "github.com/smallnest/goscapy/pkg/packet"
    "github.com/smallnest/goscapy/pkg/sendrecv"
)

// Build an IP + ICMP Echo Request packet (L3 — no Ethernet)
pkt, err := goscapy.NewIP().
    SrcIP("127.0.0.1").
    DstIP("127.0.0.1").
    Over(goscapy.NewICMP().
        Type(8).
        Code(0).
        ID(0x0001).
        Seq(1)).
    Packet()

if err != nil {
    fmt.Fatal(err)
}

// Send at L3 on loopback interface
iface := sendrecv.LoopbackName() // "lo0" on macOS, "lo" on Linux
err = sendrecv.Send(pkt, iface)
if err != nil {
    fmt.Fatal(err)
}
fmt.Println("Packet sent!")

⚡ Shortcut

go
// IPICMP returns raw bytes — use Builder API .Packet() for send/recv
pkt, _ := goscapy.NewIP().
    SrcIP("127.0.0.1").
    DstIP("127.0.0.1").
    Over(goscapy.NewICMP().Type(8).Code(0)).
    Packet()
sendrecv.Send(pkt, sendrecv.LoopbackName())
📝 Note
Send() operates at L3 (IP level) — the OS handles L2 framing (Ethernet header). If the packet contains an Ethernet layer, it is automatically skipped during build. Requires root privileges (sudo) or CAP_NET_RAW on Linux.

Recv — Receive One Packet

Open a receiver on an interface, wait for one packet, and close. Recv() blocks until a packet arrives or the timeout expires. Use errors.Is(err, sendrecv.ErrTimeout) to detect timeouts.

🔨 Example

go
import (
    "errors"
    "fmt"
    "time"

    "github.com/smallnest/goscapy/pkg/packet"
    "github.com/smallnest/goscapy/pkg/sendrecv"
)

iface := sendrecv.LoopbackName()

// Receive one packet with a 5-second timeout
pkt, err := sendrecv.Recv(iface, 5*time.Second)
if err != nil {
    if errors.Is(err, sendrecv.ErrTimeout) {
        fmt.Println("No packet received within timeout")
    } else {
        fmt.Fatal(err)
    }
    return
}

// Print the received packet layers
fmt.Println(pkt.String()) // e.g. "Ethernet / IP / ICMP"

// Access IP layer fields
ip := packet.IP(pkt)
if ip != nil {
    fmt.Printf("Source: %s → Dest: %s\n", ip.SrcIP(), ip.DstIP())
}
💡 Tip
For receiving multiple packets, use OpenReceiver() directly and call Recv() in a loop. This avoids opening and closing the raw socket for each packet.

🔧 Receiver Loop

go
// Open a persistent receiver for capturing multiple packets
rx, err := sendrecv.OpenReceiver(sendrecv.LoopbackName())
if err != nil {
    fmt.Fatal(err)
}
defer rx.Close()

// Read up to 10 packets
for i := 0; i < 10; i++ {
    pkt, err := rx.Recv(3 * time.Second)
    if err != nil {
        if errors.Is(err, sendrecv.ErrTimeout) {
            fmt.Println("Timeout, stopping.")
            break
        }
        fmt.Fatal(err)
    }
    fmt.Printf("Packet %d: %s\n", i+1, pkt.String())
}

SendRecv1 — ICMP Ping

Build an ICMP Echo Request, send it at L3, and receive the first response. SendRecv1() is ideal for request/response protocols like ICMP ping — it returns only the first matching reply.

🔨 Builder API

go
import (
    "fmt"
    "time"

    "github.com/smallnest/goscapy/pkg/goscapy"
    "github.com/smallnest/goscapy/pkg/layers"
    "github.com/smallnest/goscapy/pkg/packet"
    "github.com/smallnest/goscapy/pkg/sendrecv"
)

// Build an IP + ICMP Echo Request
pkt, err := goscapy.NewIP().
    SrcIP("127.0.0.1").
    DstIP("127.0.0.1").
    Over(goscapy.NewICMP().
        Type(layers.ICMPEchoRequest).
        Code(0).
        ID(0x0001).
        Seq(1)).
    Packet()
if err != nil {
    fmt.Fatal(err)
}

// Send and wait for the first response (3-second timeout)
sent, resp, err := sendrecv.SendRecv1(
    pkt,
    sendrecv.LoopbackName(),
    3*time.Second,
)
if err != nil {
    fmt.Fatal(err)
}

fmt.Printf("Sent:     %s\n", sent.String())
if resp != nil {
    fmt.Printf("Response: %s\n", resp.String())

    // Check if the response is an ICMP Echo Reply
    icmp := packet.ICMP(resp)
    if icmp != nil && icmp.Type() == layers.ICMPEchoReply {
        fmt.Println("✓ ICMP Echo Reply received!")
    }
}

⚡ Shortcut

go
// Quick ping using Builder .Packet() method
pkt, _ := goscapy.NewIP().
    SrcIP("127.0.0.1").
    DstIP("127.0.0.1").
    Over(goscapy.NewICMP().Type(8).Code(0)).
    Packet()

_, resp, _ := sendrecv.SendRecv1(pkt, sendrecv.LoopbackName(), 3*time.Second)
if resp != nil {
    fmt.Println("Ping reply:", resp.String())
}
📝 Note
SendRecv1() opens the receiver before sending to avoid missing fast responses (e.g. on loopback, the response can arrive before a sequentially-opened receiver is ready).

SendRecv — Collect All Responses

SendRecv() sends a packet at L3 and collects all response packets that arrive within the timeout window. Useful when you expect multiple responses (e.g. broadcast queries) or want to observe all traffic triggered by your packet.

🔨 Example

go
import (
    "fmt"
    "time"

    "github.com/smallnest/goscapy/pkg/goscapy"
    "github.com/smallnest/goscapy/pkg/layers"
    "github.com/smallnest/goscapy/pkg/sendrecv"
)

// Build ICMP Echo Request
pkt, _ := goscapy.NewIP().
    SrcIP("127.0.0.1").
    DstIP("127.0.0.1").
    Over(goscapy.NewICMP().
        Type(layers.ICMPEchoRequest).
        Code(0).
        ID(0x0001).
        Seq(1)).
    Packet()

// Send and collect all responses within 2 seconds
sent, responses, err := sendrecv.SendRecv(
    pkt,
    sendrecv.LoopbackName(),
    2*time.Second,
)
if err != nil {
    fmt.Fatal(err)
}

fmt.Printf("Sent: %s\n", sent.String())
fmt.Printf("Received %d response(s):\n", len(responses))
for i, resp := range responses {
    fmt.Printf("  [%d] %s\n", i+1, resp.String())
}

TCP SYN — Send & Recv

Construct a TCP SYN packet, send it with Send(), then wait for a SYN-ACK response with Recv(). This demonstrates the two-step send-then-receive pattern for TCP port scanning.

🔨 Builder API

go
import (
    "fmt"
    "time"

    "github.com/smallnest/goscapy/pkg/goscapy"
    "github.com/smallnest/goscapy/pkg/layers"
    "github.com/smallnest/goscapy/pkg/packet"
    "github.com/smallnest/goscapy/pkg/sendrecv"
)

// Build IP + TCP SYN packet
pkt, err := goscapy.NewIP().
    SrcIP("127.0.0.1").
    DstIP("127.0.0.1").
    Over(goscapy.NewTCP().
        SrcPort(12345).
        DstPort(80).
        Flags(layers.TCPSyn).
        Seq(1000)).
    Packet()
if err != nil {
    fmt.Fatal(err)
}

iface := sendrecv.LoopbackName()

// Step 1: Send the SYN packet
err = sendrecv.Send(pkt, iface)
if err != nil {
    fmt.Fatal(err)
}
fmt.Println("SYN sent to 127.0.0.1:80")

// Step 2: Wait for a response
resp, err := sendrecv.Recv(iface, 5*time.Second)
if err != nil {
    fmt.Fatal(err)
}

// Check if response is TCP
tcp := packet.TCP(resp)
if tcp != nil {
    flags := tcp.Flags()
    if flags&layers.TCPSyn != 0 && flags&layers.TCPAck != 0 {
        fmt.Printf("Port %d: OPEN (SYN-ACK received)\n", tcp.SrcPort())
    } else if flags&layers.TCPRst != 0 {
        fmt.Printf("Port %d: CLOSED (RST received)\n", tcp.SrcPort())
    }
}

⚡ Shortcut

go
// IPTCP shortcut for building the packet
pkt, _ := goscapy.NewIP().
    SrcIP("127.0.0.1").
    DstIP("127.0.0.1").
    Over(goscapy.NewTCP().SrcPort(12345).DstPort(80).Flags(layers.TCPSyn)).
    Packet()
sendrecv.Send(pkt, sendrecv.LoopbackName())
📝 Note
For TCP SYN scanning, SendRecv1() is often more convenient than separate Send() + Recv() calls because it opens the receiver before sending, ensuring no fast responses are missed.

Sendp — Full Ethernet Frame (L2)

Build a complete Ethernet + IP + ICMP packet and send it at L2. Unlike Send() which operates at the IP layer, Sendp() writes the entire frame — you control the Ethernet header yourself.

🔨 Builder API

go
import (
    "fmt"
    "github.com/smallnest/goscapy/pkg/goscapy"
    "github.com/smallnest/goscapy/pkg/sendrecv"
)

// Build a full Ethernet + IP + ICMP Echo Request packet (L2)
pkt, err := goscapy.NewEthernet().
    SrcMAC("00:11:22:33:44:55").
    DstMAC("ff:ff:ff:ff:ff:ff").
    Over(goscapy.NewIP().
        SrcIP("10.0.0.1").
        DstIP("10.0.0.2")).
    Over(goscapy.NewICMP().
        Type(8).
        Code(0)).
    Packet()

if err != nil {
    fmt.Fatal(err)
}

// Send the complete Ethernet frame at L2
err = sendrecv.Sendp(pkt, "eth0")
if err != nil {
    fmt.Fatal(err)
}
fmt.Println("Ethernet frame sent!")

⚡ Shortcut

go
// Build Ethernet + IP + ICMP packet using shortcut, then dissect for Sendp
raw, _ := goscapy.EtherIPICMP("ff:ff:ff:ff:ff:ff", "10.0.0.2", 8, 0)
pkt, _ := packet.Dissect(raw, packet.DissectEthernet)
sendrecv.Sendp(pkt, "eth0")
📝 Note
Send vs Sendp:
  • Send() — L3 (IP level). No Ethernet layer needed; the OS adds it.
  • Sendp() — L2 (Ethernet frame). Must include an Ethernet layer; the full frame is written to the wire.
Use Sendp() when you need custom MAC addresses, VLAN tags, or other Ethernet-level control.

Sendp — ARP Request (L2 Only)

ARP is a pure L2 protocol — it lives directly inside Ethernet with no IP layer. Sendp() is the only way to send it, because Send() requires an IP layer. Build an Ethernet + ARP "who-has" request and send it on the wire.

🔨 Builder API

go
import (
    "fmt"
    "github.com/smallnest/goscapy/pkg/goscapy"
    "github.com/smallnest/goscapy/pkg/layers"
    "github.com/smallnest/goscapy/pkg/sendrecv"
)

// Build Ethernet + ARP "who-has" request (pure L2 — no IP layer)
pkt, err := goscapy.NewEthernet().
    SrcMAC("aa:bb:cc:dd:ee:ff").
    DstMAC("ff:ff:ff:ff:ff:ff"). // broadcast
    Over(goscapy.NewARP().
        Op(layers.ARPWhoHas).
        SrcMAC("aa:bb:cc:dd:ee:ff").
        SrcIP("192.168.1.1").
        DstMAC("00:00:00:00:00:00").
        DstIP("192.168.1.100")).
    Packet()

if err != nil {
    fmt.Fatal(err)
}

// Send ARP request at L2 — must use Sendp, not Send
err = sendrecv.Sendp(pkt, "eth0")
if err != nil {
    fmt.Fatal(err)
}
fmt.Println("ARP who-has 192.168.1.100 sent!")

⚡ Shortcut

go
// EtherARP(srcMAC, dstMAC, psrc, pdst string, op uint16)
pkt, _ := goscapy.NewEthernet().
    SrcMAC("aa:bb:cc:dd:ee:ff").
    DstMAC("ff:ff:ff:ff:ff:ff").
    Over(goscapy.NewARP().
        Op(layers.ARPWhoHas).
        SrcMAC("aa:bb:cc:dd:ee:ff").
        SrcIP("192.168.1.1").
        DstMAC("00:00:00:00:00:00").
        DstIP("192.168.1.100")).
    Packet()
sendrecv.Sendp(pkt, "eth0")
📝 Note
ARP has no IP layer — it uses EtherType 0x0806 directly over Ethernet. You must use Sendp() for ARP; Send() cannot send L2-only protocols.

L2 vs L3 — When to Use Which

Understanding the difference between Send() (L3) and Sendp() (L2) is critical for choosing the right approach.

FeatureSend() — L3Sendp() — L2
Required layersIP + upperEthernet + upper
Ethernet headerOS fills in automaticallyYou must provide it
Works with ARP❌ No (ARP has no IP)✅ Yes
Works with ICMP/TCP/UDP✅ Yes✅ Yes (with Ethernet)
Custom MAC addresses❌ Not possible✅ Full control
VLAN / Dot1Q❌ Not possible✅ Supported
ImplementationAF_INET raw socketBPF (macOS) / AF_PACKET (Linux)
Use caseMost IP-based tools (ping, scan)ARP, custom L2, VLAN, MAC spoofing

        SrcIP("192.168.1.1").
        DstIP("192.168.1.2")).
    Over(goscapy.NewICMP().Type(8).Code(0)).
    Packet()

Error Handling

Production code should handle common errors: timeout, interface not found, and permission denied. Here is a robust pattern for all send/receive operations.

🔧 Timeout & Error Detection

go
import (
    "errors"
    "fmt"
    "time"

    "github.com/smallnest/goscapy/pkg/goscapy"
    "github.com/smallnest/goscapy/pkg/sendrecv"
)

// Build the packet
pkt, err := goscapy.NewIP().
    SrcIP("127.0.0.1").
    DstIP("127.0.0.1").
    Over(goscapy.NewICMP().
        Type(8).
        Code(0)).
    Packet()
if err != nil {
    fmt.Fatal("build error:", err)
}

// SendRecv1 with comprehensive error handling
_, resp, err := sendrecv.SendRecv1(pkt, sendrecv.LoopbackName(), 3*time.Second)
switch {
case err == nil && resp != nil:
    fmt.Println("Response:", resp.String())
case err == nil && resp == nil:
    fmt.Println("Sent but no response received within timeout")
case errors.Is(err, sendrecv.ErrTimeout):
    fmt.Println("Operation timed out")
default:
    fmt.Println("Error:", err)
    // Common errors:
    //   - "sendrecv: open receiver: ..." → interface does not exist
    //   - "operation not permitted"     → needs root/sudo
    //   - "no such device"             → wrong interface name
}
💡 Tip
  • Always use errors.Is(err, sendrecv.ErrTimeout) — never compare with ==.
  • Use sendrecv.LoopbackName() for portable code that works on both macOS (lo0) and Linux (lo).
  • Raw socket operations require root privileges. Run with sudo or set CAP_NET_RAW on Linux.
