Basic Sniff — Capture All Packets

Capture all packets on an interface using the callback API. The handler is called for each packet — return true to continue, false to stop.

🔨 Example

go
import (
    "fmt"
    "time"

    "github.com/smallnest/goscapy/pkg/packet"
    "github.com/smallnest/goscapy/pkg/sniff"
)

err := sniff.Sniff(sniff.SniffConfig{
    Iface:   "eth0",
    Timeout: 10 * time.Second,
}, func(pkt *packet.Packet) bool {
    // Print layer summary for each packet
    fmt.Printf("[%s] %s\n", time.Now().Format("15:04:05"), pkt.String())

    // Access specific layers
    if ip := packet.IP(pkt); ip != nil {
        fmt.Printf("  %s → %s\n", ip.SrcIP(), ip.DstIP())
    }

    return true // true = continue capturing
})
if err != nil {
    fmt.Fatal(err)
}
📝 Note
Sniff() blocks until the timeout expires, the count limit is reached, or the handler returns false. The per-read timeout is 1 second for responsiveness — ErrTimeout is handled internally and retried automatically.

Sniff with BPF Filter

Use the Filter field to capture only matching packets. The filter expression uses the same syntax as tcpdump and is compiled to kernel-level BPF instructions for maximum efficiency.

🔨 Filter String

go
import (
    "fmt"
    "time"

    "github.com/smallnest/goscapy/pkg/packet"
    "github.com/smallnest/goscapy/pkg/sniff"
)

// Capture only ICMP packets (ping)
err := sniff.Sniff(sniff.SniffConfig{
    Iface:   sendrecv.LoopbackName(),
    Filter:  "icmp",
    Timeout: 10 * time.Second,
}, func(pkt *packet.Packet) bool {
    icmp := packet.ICMP(pkt)
    if icmp != nil {
        fmt.Printf("ICMP type=%d code=%d\n", icmp.Type(), icmp.Code())
    }
    return true
})

🔧 Pre-compiled Instructions

go
// Pre-compile at startup — avoids tcpdump runtime dependency
instructions, _ := sniff.CompileFilter("tcp port 80")

// Use pre-compiled instructions (takes precedence over Filter)
err := sniff.Sniff(sniff.SniffConfig{
    Iface:        "eth0",
    Instructions: instructions, // Filter field is ignored when Instructions is set
    Timeout:      10 * time.Second,
}, func(pkt *packet.Packet) bool {
    tcp := packet.TCP(pkt)
    if tcp != nil {
        fmt.Printf("HTTP: %d → %d\n", tcp.SrcPort(), tcp.DstPort())
    }
    return true
})
📝 Note
Priority: If both Instructions and Filter are set, Instructions takes precedence. This lets you compile filters at startup and reuse them, avoiding the tcpdump runtime dependency.

Sniff — Limit Packet Count

Use the Count field to capture a fixed number of packets, then stop automatically. Combine with Filter to capture N packets matching a specific pattern.

🔨 Example

go
// Capture exactly 5 ICMP packets, then stop
err := sniff.Sniff(sniff.SniffConfig{
    Iface:  sendrecv.LoopbackName(),
    Filter: "icmp",
    Count:  5,
}, func(pkt *packet.Packet) bool {
    ip := packet.IP(pkt)
    icmp := packet.ICMP(pkt)
    fmt.Printf("%s → %s  ICMP type=%d\n",
        ip.SrcIP(), ip.DstIP(), icmp.Type())
    return true
})
💡 Tip
Set Count: 0 (the default) to capture unlimited packets. Use Count together with Timeout — sniffing stops when either limit is reached.

Sniff — with Timeout

Use the Timeout field to limit how long sniffing runs. Set Timeout: 0 (the default) for unlimited duration.

🔨 Example

go
// Capture TCP traffic for 30 seconds
err := sniff.Sniff(sniff.SniffConfig{
    Iface:   "eth0",
    Filter:  "tcp",
    Timeout: 30 * time.Second,
}, func(pkt *packet.Packet) bool {
    tcp := packet.TCP(pkt)
    fmt.Printf("TCP %d → %d flags=0x%02x\n",
        tcp.SrcPort(), tcp.DstPort(), tcp.Flags())
    return true
})
fmt.Println("Sniffing completed.")

Sniff — Stop from Handler

Return false from the handler to stop sniffing immediately. This is useful when you find the packet you're looking for.

🔨 Example

go
import (
    "fmt"
    "github.com/smallnest/goscapy/pkg/layers"
    "github.com/smallnest/goscapy/pkg/packet"
    "github.com/smallnest/goscapy/pkg/sniff"
)

// Sniff until we see an ICMP Echo Reply from a specific host
err := sniff.Sniff(sniff.SniffConfig{
    Iface:  sendrecv.LoopbackName(),
    Filter: "icmp",
    Count:  100, // safety limit
}, func(pkt *packet.Packet) bool {
    ip := packet.IP(pkt)
    icmp := packet.ICMP(pkt)

    // Check if this is an ICMP Echo Reply from 127.0.0.1
    if icmp != nil && icmp.Type() == layers.ICMPEchoReply {
        fmt.Printf("Found reply from %s!\n", ip.SrcIP())
        return false // stop sniffing immediately
    }

    fmt.Println("Waiting for reply...")
    return true // continue
})

SniffChan — Channel-Based Sniffing

SniffChan() returns a channel that delivers captured packets and a stop function. This is more ergonomic than callbacks when integrating with Go's select and channel patterns.

🔨 Example

go
import (
    "fmt"
    "time"

    "github.com/smallnest/goscapy/pkg/sniff"
)

// Start sniffing — returns a channel and a stop function
ch, stop := sniff.SniffChan(sniff.SniffConfig{
    Iface:   "eth0",
    Filter:  "tcp port 80",
    Count:   10,
    Timeout: 30 * time.Second,
})
defer stop() // Always call stop to release resources

// Range over the channel to receive packets
for pkt := range ch {
    fmt.Println(pkt.String())
}
fmt.Println("Channel closed — sniffing complete.")

🔧 Stop Early with Select

go
ch, stop := sniff.SniffChan(sniff.SniffConfig{
    Iface:  "eth0",
    Filter: "tcp port 80",
})

// Use select for custom stop conditions
timeout := time.After(5 * time.Second)
for {
    select {
    case pkt, ok := <-ch:
        if !ok {
            fmt.Println("Done.")
            return
        }
        fmt.Println(pkt.String())
    case <-timeout:
        fmt.Println("Custom timeout reached.")
        stop()
    }
}

Combined — Count + Timeout + Filter

All config options can be combined. Sniffing stops when any condition is met: count limit, timeout, or handler returns false.

🔨 Full Example

go
package main

import (
    "fmt"
    "os"
    "time"

    "github.com/smallnest/goscapy/pkg/layers"
    "github.com/smallnest/goscapy/pkg/packet"
    "github.com/smallnest/goscapy/pkg/sendrecv"
    "github.com/smallnest/goscapy/pkg/sniff"
)

func main() {
    // Pre-compile filter at startup
    instructions, err := sniff.CompileFilter("tcp port 80 or tcp port 443")
    if err != nil {
        fmt.Fprintf(os.Stderr, "compile filter: %v\n", err)
        os.Exit(1)
    }

    // Capture up to 50 HTTP/HTTPS packets for 1 minute
    captured := 0
    err = sniff.Sniff(sniff.SniffConfig{
        Iface:        sendrecv.LoopbackName(),
        Instructions: instructions,
        Count:        50,
        Timeout:      60 * time.Second,
    }, func(pkt *packet.Packet) bool {
        captured++
        ip := packet.IP(pkt)
        tcp := packet.TCP(pkt)

        // Classify traffic direction
        dir := "→"
        if tcp.DstPort() == 80 || tcp.DstPort() == 443 {
            dir = "→ request"
        } else {
            dir = "← response"
        }

        // Check for SYN flag
        flags := tcp.Flags()
        if flags&layers.TCPSyn != 0 && flags&layers.TCPAck == 0 {
            fmt.Printf("[%d] %s:%d %s %s:%d SYN\n",
                captured, ip.SrcIP(), tcp.SrcPort(), dir, ip.DstIP(), tcp.DstPort())
        } else {
            fmt.Printf("[%d] %s:%d %s %s:%d\n",
                captured, ip.SrcIP(), tcp.SrcPort(), dir, ip.DstIP(), tcp.DstPort())
        }

        return true
    })
    if err != nil {
        fmt.Fatal(err)
    }
    fmt.Printf("Captured %d packets.\n", captured)
}
